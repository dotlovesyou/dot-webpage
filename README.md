# Dot - Pippin's Ladybug Friend

A simple website for Dot, the ladybug character from Pippin's Wobbly Woods.

## Links

- Twitter: [@DotlovesyouSOL](https://x.com/DotlovesyouSOL)
- GitHub: [dotlovesyou](https://github.com/dotlovesyou)

## Pages

- **Home** - Introduction to Dot and the Spotted Gardens
- **Token** - Info about the $DOT token (CA coming soon)
- **Ladybug** - Dot's backstory and her friendship with Pippin
- **Gallery** - Collection of tweets from [@pippinlovesyou](https://x.com/pippinlovesyou) mentioning Dot
- **Shop** - Coming soon

## Gallery

The gallery page features embedded tweets from Pippin's Twitter account where he mentions his ladybug friend Dot. These are official tweets showing the lore connection between Pippin and Dot.

## Tech

Static HTML/CSS/JS site. Uses Twitter's embed widget for the gallery.
